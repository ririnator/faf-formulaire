# Temporary Files

This directory contains temporary files generated during development or testing.

## Files
- `cookies.txt` - Browser session cookies for testing (should be ignored by Git)
- `migration-report-1755199620215.json` - Migration validation report
- `validation-phase-1.json` - Phase 1 validation results

## Usage
This directory is for temporary files that shouldn't be committed to the repository. Files here are typically generated during development, testing, or debugging sessions.

**Note**: Add any temporary files you want to keep locally but not commit to this directory.
