# Backups MongoDB

Ce dossier contient les sauvegardes MongoDB générées par le script `backup-mongodb.js`.

## Format des fichiers

`mongodb-backup-{timestamp}.json`

## Sécurité

⚠️ **Ne JAMAIS commiter ce dossier** : Contient des données personnelles

## Utilisation

- Sauvegarde de sécurité avant migration
- Validation post-migration
- Rollback si nécessaire
- Archive long terme
